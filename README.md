## build the project

To start the server 

create a .env file in server and populate CONNECTION_URL and PORT
PORT = 5000
CONNECTION_URL= mongodburl

```bash
  cd server
```
```bash
  npm start
```

To start the frontend 
go to client1
first install all dependencies in package.json
```bash
  npm install
```

```bash
  npm start
```
